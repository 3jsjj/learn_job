
# 基于 Abaqus 的触觉压力场重建数据生成系统设计说明

## 1. 系统目标

本系统用于替代原 MATLAB 代码中“随机高斯函数生成训练数据”的部分。

原始代码的数据生成方式是：

```text
随机高斯参数
      ↓
生成 21×21 高斯压力场
      ↓
提取 9 个传感器点
      ↓
训练 MLP
```

这种方式的问题是：MLP 实际上只是在学习高斯模型，不能反映真实接触、材料变形、边缘效应和复杂压力分布。

改进后的系统目标是：

```text
Abaqus 有限元仿真
      ↓
获得真实接触压力场
      ↓
提取 9 个传感器点数据作为输入
      ↓
提取 21×21 压力场作为标签
      ↓
训练 MLP 重建真实压力场
```

---

## 2. 总体系统框架

```text
参数设计模块
      │
      ▼
Abaqus 自动建模模块
      │
      ▼
批量仿真模块
      │
      ▼
ODB 结果提取模块
      │
      ▼
数据整理模块
      │
      ▼
MATLAB / Python 神经网络训练模块
```

系统最终生成的数据格式为：

```text
X_data: N × 9
Y_data: N × 441
```

其中：

- `X_data`：9 个压力传感器的响应值；
- `Y_data`：21×21 网格上的真实压力场；
- `N`：仿真样本数量。

---

# 3. Abaqus 有限元模型设计

## 3.1 模型结构

建议建立如下模型：

```text
刚性压头
   ↓
柔性弹性层 / 硅胶层
   ↓
压力传感器层
   ↓
刚性基底
```

每一层作用如下：

| 部件 | 作用 |
|---|---|
| 刚性压头 | 模拟外部接触物体 |
| 柔性弹性层 | 模拟触觉皮肤或封装材料 |
| 压力传感器层 | 用于提取 9 个传感器区域的力/压力 |
| 刚性基底 | 提供固定支撑 |

---

## 3.2 几何尺寸建议

如果当前传感器区域为 `20 mm × 20 mm`，可以设置：

| 参数 | 建议值 |
|---|---|
| 触觉阵列尺寸 | 20 mm × 20 mm |
| 弹性层厚度 | 2~5 mm |
| 基底厚度 | 1~2 mm |
| 压头半径 | 2~8 mm |
| 传感器数量 | 3 × 3 |
| 传感器间距 | 10 mm |
| 输出网格 | 21 × 21 |

9 个传感器位置与原 MATLAB 代码一致：

```matlab
sensor_points = [
    0, 0;
    10, 0;
    20, 0;
    0, 10;
    10, 10;
    20, 10;
    0, 20;
    10, 20;
    20, 20
];
```

# 4. 参数扫描方案

## 4.1 需要扫描的参数

建议至少扫描以下参数：

| 参数 | 符号 | 建议范围 |
|---|---|---|
| 接触中心 x 坐标 | `cx` | 0~20 mm |
| 接触中心 y 坐标 | `cy` | 0~20 mm |
| 压入力 | `F` | 5~50 N |
| 压头半径 | `R` | 2~8 mm |
| 弹性层模量 | `E` | 可选 |
| 摩擦系数 | `mu` | 可选 |

核心参数是：

```text
cx, cy, F, R
```

---

## 4.2 样本数量估算

例如：

| 参数 | 取值数量 |
|---|---|
| x 方向位置 | 21 |
| y 方向位置 | 21 |
| 压入力 | 10 |
| 压头半径 | 4 |

总样本数为：

```text
21 × 21 × 10 × 4 = 17640 组
```

如果计算量较大，可以先降低分辨率：

| 参数 | 取值数量 |
|---|---|
| x 方向位置 | 11 |
| y 方向位置 | 11 |
| 压入力 | 6 |
| 压头半径 | 3 |

总样本数为：

```text
11 × 11 × 6 × 3 = 2178 组
```

建议流程：

```text
先跑 500~2000 组验证流程
再扩展到 10000~50000 组正式训练
```

# 5. Abaqus 批量仿真脚本设计

## 5.1 主控脚本逻辑

Abaqus 支持 Python 脚本自动建模、提交任务和导出结果。

主控脚本结构如下：

```python
# run_batch_simulation.py

xs = range(0, 21, 1)
ys = range(0, 21, 1)
forces = [5, 10, 15, 20, 25, 30, 35, 40, 45, 50]
radii = [2, 4, 6, 8]

sample_id = 0

 for cx in xs:
    for cy in ys:
        for force in forces:
            for radius in radii:

                sample_id += 1

                create_model(
                    sample_id=sample_id,
                    cx=cx,
                    cy=cy,
                    force=force,
                    radius=radius
                )

                submit_job(sample_id)

                extract_result(sample_id)
```

注意：正式脚本中 `for cx in xs:` 前面不能有多余空格。

---

## 5.2 每组仿真需要修改的内容

每一组仿真主要改变：

1. 压头中心位置；
2. 压头半径；
3. 施加载荷；
4. Job 名称；
5. 输出文件名。

例如：

```text
sample_000001:
cx = 0 mm
cy = 0 mm
F  = 5 N
R  = 2 mm

sample_000002:
cx = 0 mm
cy = 0 mm
F  = 5 N
R  = 4 mm
```

# 6. Abaqus 输出结果提取

## 6.1 需要提取的数据

每一组仿真需要提取两类数据。

第一类：9 个传感器位置的压力值。

```text
sensor_1, sensor_2, ..., sensor_9
```

第二类：21×21 网格压力场。

```text
pressure_1, pressure_2, ..., pressure_441
```

最终一条样本数据可以表示为：

```text
sample_id,
cx, cy, force, radius,
sensor_1, ..., sensor_9,
pressure_1, ..., pressure_441
```

---

## 6.2 压力场提取方式

Abaqus 中可使用以下结果变量：

| 变量 | 含义 | 适用情况 |
|---|---|---|
| `CPRESS` | 接触压力 | 接触面压力分布 |
| `S` | 应力 | 材料内部应力 |
| `RF` | 反力 | 传感器区域合力 |
| `U` | 位移 | 变形分析 |

对于触觉压力场重建，优先使用：

```text
CPRESS
```

如果传感器实际测量的是某区域的合力，则可对传感器区域节点或单元做积分，得到 9 个传感器响应。

---

## 6.3 传感器值提取方法

有两种常见方案。

### 方案 A：点采样

直接读取 9 个传感器中心附近的压力值。

优点：

```text
简单，容易实现
```

缺点：

```text
对网格敏感，容易受局部噪声影响
```

### 方案 B：区域平均 / 区域积分

对每个传感器覆盖区域内的压力进行平均或积分。

优点：

```text
更接近真实传感器输出
更稳定
更适合论文
```

推荐使用方案 B。

传感器值可以定义为：

```text
sensor_i = 第 i 个传感器区域内 CPRESS 的平均值
```

或：

```text
sensor_i = 第 i 个传感器区域内压力积分得到的合力
```

# 9. MATLAB 训练代码如何修改

原代码中第 7 步为：

```matlab
N = 3000;
X_data = zeros(N, 9);
Y_data = zeros(N, n_grid);

for k = 1:N
    % 随机生成高斯压力场
end
```

使用 Abaqus 数据后，应改为：

```matlab
X_table = readtable('X_data.csv');
Y_table = readtable('Y_data.csv');

X_data = table2array(X_table(:, 2:end));
Y_data = table2array(Y_table(:, 2:end));

N = size(X_data, 1);
```

后面的归一化、训练集划分、网络结构、训练过程可以基本保持不变：

```matlab
mu_X = mean(X_data, 1);
std_X = std(X_data, 0, 1) + eps;

mu_Y = mean(Y_data, 1);
std_Y = std(Y_data, 0, 1) + eps;

X_norm = (X_data - mu_X) ./ std_X;
Y_norm = (Y_data - mu_Y) ./ std_Y;
```

# 10. 数据质量控制

## 10.1 删除异常样本

需要剔除以下样本：

```text
仿真未收敛
接触失败
压力场全为 0
局部压力异常过大
ODB 文件损坏
输出数据维度不正确
```

---

## 10.2 保证数据覆盖均匀

训练数据应覆盖：

```text
中心接触
边缘接触
角落接触
轻载接触
重载接触
小压头接触
大压头接触
```

否则 MLP 可能只在部分区域预测效果好。

---

## 10.3 训练集与测试集划分

不要只随机划分，还应额外设置外推测试。

例如：

| 测试类型 | 目的 |
|---|---|
| 随机测试 | 验证总体预测能力 |
| 边缘区域测试 | 验证边界效应 |
| 未见载荷测试 | 验证力大小泛化 |
| 未见半径测试 | 验证接触面积泛化 |
| 实验数据测试 | 验证真实可用性 |

# 12. 与原高斯代码的关系

原始高斯代码可以保留，但应作为对比基线。

推荐论文中设置两种方法：

```text
Baseline：加权质心 + 高斯重建
Proposed：Abaqus 数据训练的 MLP 重建
```

这样逻辑更加合理：

```text
高斯方法：简单、可解释，但表达能力有限
MLP 方法：基于仿真数据学习复杂压力分布
```

---

# 13. 论文方法部分可用表述

可以写成：

> 为获得神经网络训练所需的大规模压力场数据，本文基于 Abaqus 建立了柔性触觉阵列的有限元模型。通过参数化改变接触位置、法向载荷和压头半径，批量生成不同接触工况下的压力分布。每组仿真中，提取 3×3 传感器位置处的压力响应作为网络输入，并将接触表面 21×21 网格上的压力分布作为监督标签。由此构建输入维度为 9、输出维度为 441 的触觉压力场重建数据集，用于训练多层感知机模型。

---

# 14. 注意事项

1. Abaqus 数据生成比高斯数据慢很多，必须使用 Python 批处理。
2. 初期不要直接跑几万组，先用几十组验证提取脚本是否正确。
3. 网格划分要保证 21×21 输出点能稳定插值。
4. 传感器值建议用区域平均或区域积分，而不是单点采样。
5. 压力单位要统一，例如 N、MPa、mm。
6. 训练集和测试集不能只看随机划分，还要单独测试边缘、角落和未见参数。
7. 高斯方法可以作为 baseline，不建议作为 MLP 标签来源。
8. 如果后续有真实实验数据，应使用实验数据校正 Abaqus 模型。

---

# 15. 最终建议

推荐最终系统采用：

```text
Abaqus 仿真数据 + MLP 重建
```

而不是：

```text
高斯模拟数据 + MLP 重建
```

原因是：

```text
高斯模拟数据只能让 MLP 学会高斯模型；
Abaqus 仿真数据可以让 MLP 学习材料变形、边缘效应和复杂接触规律。
```

因此，本系统更适合作为论文或工程应用中的正式数据生成方案。
