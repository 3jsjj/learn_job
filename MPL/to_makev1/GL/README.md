# 同结构概率反演版说明

## 1. 与原 MLP 代码的关系

本版本功能与原代码一致：

```text
9个传感器输入
      ↓
重建21×21力场
      ↓
输出结果表
      ↓
检查9个传感器位置误差
      ↓
绘制三维图、二维等高线图、对比图
```

区别是：

```text
原代码：
构造训练数据 → 训练 MLP → predict(net)

本代码：
构造候选数据库 → 计算后验概率 → MAP/后验均值重建
```

## 2. 文件说明

```text
main_probability_inverse_reconstruction.m
主程序，结构与原 MLP 主程序基本一致。

bayesian_inverse_predict.m
概率反演核心函数。

estimate_contact_params.m
高斯参数估计函数。

gaussian_fit_error.m
高斯半径拟合误差函数。
```

## 3. 运行方式

将四个 `.m` 文件放在同一文件夹，运行：

```matlab
main_probability_inverse_reconstruction
```

## 4. 方法含义

本方法不是训练神经网络。

它会先生成大量候选压力场：

```text
候选1：9点响应 + 441点力场
候选2：9点响应 + 441点力场
...
候选N：9点响应 + 441点力场
```

然后拿当前输入的9点传感器值与所有候选的9点响应进行比较。

越接近当前观测的候选，后验概率越高。

最终输出：

```text
MAP压力场：
后验概率最高的候选力场。

后验均值压力场：
所有候选力场按后验概率加权平均。

后验标准差：
每个位置的不确定性。
```

## 5. 后续接入 Abaqus 数据

如果以后不用高斯随机生成候选库，而是使用 Abaqus 数据，只需要替换主程序第7步：

```matlab
X_data = Abaqus生成的 N×9 数据;
Y_data = Abaqus生成的 N×441 数据;
```

第8步之后的概率反演流程不需要改。
