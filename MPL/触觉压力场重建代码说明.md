# MATLAB 触觉压力场重建代码说明

## 1. 项目目标

本代码演示如何利用 **9 个压力传感器** 重建 **21×21
的二维压力场**，并比较两种方法：

1.  加权质心 + 高斯模型重建
2.  MLP（多层感知机）重建

------------------------------------------------------------------------

## 2. 整体流程

``` text
9个传感器数据
      │
      ├── 方法一
      │     estimate_contact_params()
      │           │
      │           ▼
      │   估计接触中心(cx、cy)
      │   估计接触半径(r)
      │   估计峰值(amp)
      │   估计背景力(F_base)
      │           │
      │           ▼
      │     高斯函数重建压力场
      │
      └── 方法二
            生成训练数据
                  │
                  ▼
              训练 MLP
                  │
                  ▼
          根据9点预测21×21压力场
```

------------------------------------------------------------------------

# 3. 为什么代码里会随机生成高斯参数？

代码中的：

``` matlab
cx_true = 20 * rand();
cy_true = 20 * rand();

sigma_true = 1.5 + 5.0 * rand();
amp_true = 800 + 6000 * rand();
F_base_true = 500 + 800 * rand();
```

**不是用于预测当前接触点。**

它们的作用是**随机生成训练样本**。

每一次循环都会随机产生：

-   接触中心 `(cx_true, cy_true)`
-   接触区域大小 `sigma_true`
-   峰值压力 `amp_true`
-   背景压力 `F_base_true`

然后利用二维高斯函数生成一个完整压力场：

``` math
F(x,y)=F_{base}+A\exp\left(
-\frac{(x-c_x)^2+(y-c_y)^2}{2\sigma^2}
\right)
```

这个压力场被当作"真实标签"。

随后：

-   输入：9 个传感器位置的数据
-   输出：441 个网格点压力

用于训练 MLP。

------------------------------------------------------------------------

# 4. 为什么还要训练 MLP？

这里容易误解。

实际上：

-   **高斯模型**负责生成训练数据。
-   **MLP**负责学习"9 个传感器 → 完整压力场"的映射。

因此：

> MLP 并没有学习真实物理规律，而是在学习高斯模型。

------------------------------------------------------------------------

# 5. 如果目标只是预测接触点，需要 MLP 吗？

通常 **不需要**。

直接利用：

``` text
9个传感器
      ↓
加权质心
      ↓
估计(cx,cy)
      ↓
高斯拟合
```

即可获得：

-   接触中心
-   接触半径
-   峰值
-   背景力

速度更快、解释性更强。

------------------------------------------------------------------------

# 6. 当前代码最大的局限

训练数据全部来自二维高斯模型，因此：

``` text
高斯模型
    ↓
生成训练标签
    ↓
MLP学习高斯模型
```

最终 MLP 学到的仍然是高斯分布。

因此它难以处理：

-   非规则接触
-   多接触点
-   非对称压力分布
-   材料非线性
-   边界效应

------------------------------------------------------------------------

# 7. 更合理的研究流程

建议使用：

``` text
Abaqus仿真 / 实验标定
          ↓
真实441点压力场
          ↓
提取9个传感器数据
          ↓
训练MLP
          ↓
预测真实压力场
```

这样 MLP 学习的是实际物理规律，而不是人为设定的高斯模型。

------------------------------------------------------------------------

# 8. 总结

本代码更适合作为 **Demo**：

-   演示如何由 9 个传感器重建二维压力场；
-   比较高斯重建与 MLP 重建。

若用于论文或工程，建议使用 Abaqus
或实验测量数据替代高斯生成数据，以提高模型的真实性和学术价值。
